## 图片查看小工具

### Installation

```shell
npm install canvas-image-viewer
```

In browser:

```html
<link  href="/dist/index.css" rel="stylesheet">
<script src="/dist/index.js"></script>
```

### Usage

```js
    new ImageViewer(document.getElementById('app'), {
        src: 'https://www.w3school.com.cn/i/eg_tulip.jpg'
    })
```
